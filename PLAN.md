## Тема исследования

**«Социальное доверие и готовность к кооперации: анализ детерминант на основе World Values Survey»**

Связка социологии и теории игр: вопросы WVS (доверие, оправданность обмана) — это поведенческие прокси для классических концепций теории игр (Prisoner's Dilemma, free-rider problem).

**Исследовательский вопрос:** какие демографические и культурные факторы определяют склонность людей к кооперативному поведению в обществе?

# ФАЗА 3 — Часть III: Вероятность и двумерные переменные

**Цель:** Определить 2 события, посчитать вероятности и проверить независимость.

**Требование из PDF (Część III):**
> - Zdefiniować co najmniej 2 zdarzenia
> - Obliczyć: P(A), P(B), P(A|B)
> - sprawdzić niezależność zdarzeń

### Определения событий:

- **A = «Респондент доверяет большинству людей»** (`trust == 1`)
  → прокси для готовности к кооперации в Prisoner's Dilemma
- **B = «Респондент считает безбилетный проезд никогда не оправданным»** (`cheat_fare == 1`)
  → прокси для отказа от поведения free-rider

### Задачи:
1. Посчитать вероятности:
   ```r
   PA  <- dt[, mean(trust == 1)]
   PB  <- dt[, mean(cheat_fare == 1)]
   PAB <- dt[, mean(trust == 1 & cheat_fare == 1)]
   PA_given_B <- PAB / PB
   ```
2. Проверить независимость:
   - Сравнить `PAB` с `PA * PB`
   - Если равны (с точностью до округления) → независимы
   - Если нет → зависимы (указать направление связи)
3. Подтвердить тестом χ²:
   ```r
   chisq.test(table(dt$trust, dt$cheat_fare))
   ```
4. **Интерпретация:** связать с теорией игр — обсудить, что зависимость событий подтверждает существование «кооперативного типа личности».

# ФАЗА 4 — Часть IV: Wnioskowanie statystyczne (CI + гипотезы)

**Цель:** Построить доверительный интервал и проверить ≥5 гипотез.

**Требование из PDF (Część IV):**
> - **Przedziały ufności:** 95% CI dla średniej
> - **Testowanie hipotez:** ≥3 hipotezy na zmiennej liczbowej (t-test/ANOVA + nieparametryczne) + ≥2 hipotezy na zmiennych jakościowych (np. Chi²)
> - **Pamiętaj o założeniach testów**
> - Podaj p-value, interpretuj przy alpha = 0.05

### 4.1. Доверительный интервал

95% CI для среднего `life_sat`:
```r
t.test(dt$life_sat)$conf.int
```
Интерпретация: «с вероятностью 95% истинное среднее значение лежит в интервале [a, b]».

### 4.2. Числовые гипотезы (минимум 3)

**Гипотеза H1: пол vs удовлетворённость жизнью**
- H₀: средняя `life_sat` одинакова у мужчин и женщин
- Тест: **t-Студента** (двухвыборочный)
- Непарам. альтернатива: **Wilcoxon (Mann-Whitney U)**
- Проверка предпосылок: нормальность (Shapiro-Wilk), равенство дисперсий (Levene)

**Гипотеза H2: уровень образования vs доверие**
- H₀: уровень `trust` (как числовой) одинаков во всех группах образования
- Тест: **ANOVA**
- Непарам. альтернатива: **Kruskal-Wallis**

**Гипотеза H3: страна vs толерантность к обману (`cheat_taxes`)**
- H₀: средняя `cheat_taxes` одинакова во всех странах (выбрать 3-5 стран для упрощения)
- Тест: **ANOVA**
- Непарам.: **Kruskal-Wallis**

### 4.3. Качественные гипотезы (минимум 2)

**Гипотеза H4: пол vs доверие**
- H₀: пол и `trust` независимы
- Тест: **χ² независимости**

**Гипотеза H5: важность религии vs толерантность к обману**
- Разбить `cheat_taxes` на категории: low/medium/high
- H₀: уровень религиозности и толерантность к обману независимы
- Тест: **χ² независимости**

### 4.4. Для каждого теста обязательно:
- Сформулировать H₀ и H₁
- Проверить предпосылки
- Привести значение статистики + df + p-value
- Сравнить p-value с α = 0.05
- Текстовая интерпретация в формате PDF:
  > _«Test t-Studenta wykazał istotną różnicę w poziomie X między A i B, t(df)=val, p=val.»_

### Результат фазы:
- 95% CI для life_sat с интерпретацией
- Таблица 5 гипотез: H₀, тест, статистика, p-value, вывод
